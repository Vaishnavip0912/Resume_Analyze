# 🚀 AI Resume Analyzer (ATS + AI Agent System)

## 📌 Overview

An AI-powered Resume Analyzer that evaluates candidate resumes against job descriptions using a multi-step AI Agent pipeline.
The system performs **PDF parsing, NLP-based skill extraction, ATS scoring, and intelligent recommendations** to simulate real-world hiring workflows.

---

## 🎯 Key Features

✅ 📄 Upload Resume (PDF)
✅ 🧠 NLP-based Skill Extraction
✅ 🤖 AI Agent Pipeline (Multi-step processing)
✅ 📊 ATS Score Calculation
✅ 🎯 Job Description Matching
✅ ⚠️ Missing Skills Detection
✅ 💡 Smart Suggestions for Improvement

---

## 🧠 System Architecture

This project is designed using a **modular AI Agent architecture**:

1. **Skill Extraction Agent**
   Extracts relevant technical skills from resume text

2. **Matching Agent**
   Compares resume skills with job requirements

3. **Evaluation Agent**
   Calculates ATS score based on skill match

4. **Suggestion Agent**
   Generates actionable recommendations

👉 This simulates how modern AI-driven recruitment systems work.
